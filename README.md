# PWNC4M
This is the source code about ctf team of University of Camerino. <br />
## Install repo
### Https
<pre>
git clone https://github.com/PWNC4M/PWNC4M.git
</pre>
### Ssh
<pre>
git clone git@github.com:PWNC4M/PWNC4M.git
</pre>
## Build
<pre>
jekyll build
</pre>
## Starting
<pre>
jekyll serve --livereload
</pre>
